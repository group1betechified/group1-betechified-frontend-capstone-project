# Express backend + HTML

## 📌 Overview
This is a simple Express.js backend with CRUD operations and a frontend (HTML/CSS) served from the `public` folder. Notice I moved html, css and images to the folder 'public'

---

## 🛠 Requirements
- Node.js (v16 or higher recommended)
- npm (comes with Node.js)

---

## ⚙️ Installation
1. Clone the repository
   ```bash
   cd express-backend

2. run:
    npm install

3. Start the Express server:
    node server.js

4. open your browser:
    http://localhost:3000
